## 概要

このリポジトリは、GitHub Actions の様々な機能をテストし、ベストプラクティスを研究するためのラボラトリーです。

主な内容:
- GitHub Actions の基本的な使い方
- 他の CI プラットフォーム（CircleCI、Azure Pipeline、Jenkins）との比較
- セキュリティのベストプラクティス
- 高度な使用例とチートシート

詳細なドキュメント、ワークフローの例、コードサンプルについては、[English README](README.md) をご覧ください。
